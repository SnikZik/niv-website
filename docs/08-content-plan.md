# 08 — תוכנית תוכן, תבניות עמוד וקישור פנימי

## היררכיית תוכן
```
בית  →  עמוד דגל "מנעולן לדלתות בירושלים"
         ├── 10 עמודי שירות
         ├── עמוד־על "מנעולן בירושלים" → אזורים → שירות×אזור
         └── 15 מדריכים (תומכי SEO, מזרימים קישורים לשירותים)
```

## תבניות עמוד (page templates)

| תבנית | קובץ | סקשנים (מלמעלה למטה) |
|-------|------|----------------------|
| בית | `front-page.php` | הירו חירום → CTA דביק → כרטיסי שירות → למה ניב → הסבר חירום → גריד אזורים → סקשן מילת מפתח → עדויות* → FAQ → תצוגת מדריכים → CTA סופי → footer |
| שירות | `single-service.php` | פירורים → הירו + CTA → הסבר שירות → מתי להתקשר → איך התהליך → גורמי מחיר → אמון → FAQ → שירותים קשורים → אזורים קשורים → CTA |
| ארכיון שירותים | `archive-service.php` | כותרת → גריד כרטיסי שירות לפי קטגוריה → CTA |
| אזור | `single-service_area.php` | פירורים → הירו "מנעולן ב{אזור}" + CTA → אינטרו מקומי → שירותים באזור → קישורים לשירות×אזור → אזורים סמוכים → FAQ אזורי → אמון → CTA |
| שירות×אזור | `single-service_area_page.php` | פירורים → הירו H1 ייחודי + CTA → אינטרו ייחודי → הקשר מקומי → פסקת שירות → FAQ → קישורים (שירות אב, אזור אב, סמוכים, שירותים דומים) → CTA |
| מדריך | `single.php` | פירורים → כותרת → תוכן → CTA באמצע → קישורים לשירותים רלוונטיים → מדריכים קשורים → CTA |
| כללי | `page.php` | Flexible sections (ACF) |

*עדויות — רק אם יש ביקורות אמת.

## חוקי קישור פנימי (אין עמודים יתומים)

**עמוד שירות** מקשר ל: בית · קשר · אזורי שירות רלוונטיים · 3 שירותים קשורים · 2 מדריכים.
**עמוד אזור** מקשר ל: בית · עמוד הדגל · שירותים באזור · אזורים סמוכים · קשר.
**שירות×אזור** מקשר ל: שירות אב · אזור אב · אזורים סמוכים · שירותים דומים · קשר.
**Footer** (בכל עמוד): שירותים ראשיים · אזורים ראשיים · קשר · נגישות · פרטיות · מדריכים.

התבניות ממלאות קישורים חסרים אוטומטית (מאותה קטגוריה/אזור) — ראה `02` ברירות מחדל.

## H1 + מבנה H2 לעמודי הליבה

**בית** — H1: `מנעולן לדלתות בירושלים`
H2: השירותים שלנו · למה ניב המנעולן · נתקעתם בחוץ? כך זה עובד · אזורי שירות בירושלים והסביבה · שאלות נפוצות · מדריכים.

**עמוד דגל** — H1: `מנעולן לדלתות בירושלים`
H2: מתי קוראים למנעולן לדלת · סוגי דלתות שאנחנו פותחים · איך נפתחת דלת בלי נזק · מה משפיע על המחיר · אזורי שירות · שאלות נפוצות.

**שירות (דוגמה — החלפת צילינדר)** — H1: `החלפת צילינדר בירושלים`
H2: מתי צריך להחליף צילינדר · סוגי צילינדרים · איך מתבצעת ההחלפה · מה משפיע על המחיר · אזורי שירות · שאלות נפוצות.

**אזור (דוגמה — גבעת זאב)** — H1: `מנעולן בגבעת זאב`
H2: שירותי מנעולן בגבעת זאב · תקלות נפוצות באזור · אזורים סמוכים · שאלות נפוצות.

## 15 מאמרי מדריכים (פתיחה)

1. איך לבחור מנעולן בירושלים בלי ליפול על חאפר
2. דלת נטרקה והמפתח בפנים — מה עושים
3. כמה עולה פריצת דלת בירושלים
4. מתי חייבים להחליף צילינדר
5. פריצת דלת ממ״ד — מה חשוב לדעת
6. איך לזהות מנעולן אמין
7. מה עושים אם המפתח נשבר בתוך הדלת
8. החלפת צילינדר בדירה שכורה — למה זה חשוב
9. מנעול חכם לדלת — יתרונות וחסרונות
10. איך לשמור על דלת הכניסה בבית ישן בירושלים
11. מה משפיע על מחיר מנעולן
12. האם אפשר לפתוח דלת בלי נזק
13. מתי צריך תיקון מנגנון ולא רק החלפת צילינדר
14. מנעולן בלילה — מה חשוב לבדוק לפני שמזמינים
15. דלת פלדלת לא נפתחת — סיבות ופתרונות

כל מאמר מקשר ל־2+ עמודי שירות רלוונטיים + עמוד אזור אחד.

## Workflow כתיבה — 3 שלבים (חובה לכל עמוד)

**שלב 1 — אסטרטג SEO:** כוונת חיפוש · מילת מפתח ראשית/משניות · קישורים פנימיים · H1 · מבנה H2 · שאלות FAQ · זווית CTA · רלוונטיות מקומית · מה אסור להגזים.

**שלב 2 — קופירייטר עברי ממיר:** כותב טבעי, ישראלי, קצר, ישיר. מדבר למי שתקוע מחוץ לדלת עכשיו. בלי stuffing, בלי "בעידן המודרני / פתרון מקיף / ללא פשרות", בלי להמציא עובדות.

**שלב 3 — פרופסור/עורך עברי:** בודק עברית טבעית, בלי ניסוח AI, בלי חזרות, פיסוק ומרכאות נכונים, קצב, טון ישראלי מקומי, CTA ברור, מונחי SEO טבעיים, בלי הבטחות שווא. משכתב אם צריך.
