# 07 — צ'קליסט השקה + QA

- [ ] אין טענות מזויפות (ניסיון, משטרה, ביקורות, מחירים, זמן הגעה).

## SEO
- [ ] H1 יחיד וייחודי בכל עמוד.
- [ ] כל title קיים, 55–60 תווים, **ייחודי** (סריקת כפילויות).
- [ ] כל meta description קיים, 130–145 תווים, **ייחודי**.
- [ ] פירורי לחם עובדים בכל עמוד פנימי.
- [ ] קישור פנימי מלא — אין עמודים יתומים.
- [ ] canonical נכון; עמודים דקים = noindex/draft.
- [ ] Sitemap XML נוצר וכולל CPTs; מחריג noindex.
- [ ] robots.txt נכון + מצביע על sitemap.
- [ ] סכמות תקינות ב־Rich Results Test (Locksmith, Service, FAQ, Breadcrumb).
- [ ] alt לכל תמונה משמעותית.

## המרה ו־UX
- [ ] טלפון ב־header עובד (tel:).
- [ ] CTA דביק בנייד עובד (call + WhatsApp).
- [ ] כפתור WhatsApp פותח שיחה עם המספר הנכון.
- [ ] טופס עובד — נשלח, מגיע ליעד, מציג הצלחה/שגיאה.
- [ ] מספר טלפון גלוי תוך שנייה בנייד (לא מוסתר מאחורי טופס).
- [ ] כל CTA משתמש בטקסט ברור.

## טכני / ביצועים

- [ ] אין CLS (מידות תמונה מוגדרות).

- [ ] גיבוי לפני עלייה; Search Console + sitemap הוגשו.
