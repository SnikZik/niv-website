# ניב המנעולן — אתר וורדפרס

אתר SEO + המרה למנעולן בירושלים. נבנה כ־Custom Theme קליל מבוסס ACF, RTL, ללא Elementor.

**מילת מפתח ראשית:** מנעולן לדלתות בירושלים

## עקרונות ליבה (מספר המותג)

1. כל מסך מוביל לפעולה אחת ברורה — **להתקשר**.
2. מספר טלפון בולט בראש כל עמוד + כפתור חיוג קבוע בנייד.
3. אדום `#C0392B` שמור לפעולות בלבד, לא לקישוט.
4. הרבה מרחב לבן, טקסט גדול וקריא.
5. עדות אמת ואזורי שירות במקום הבטחות ריקות.
6. תמונות אמת בלבד — לא סטוק, לא דלתות שבורות מבוימות.
7. הטקסט אנושי, ישראלי, לא נשמע כמו AI.

## איך משתמשים

התמה נועדה להיכנס ל־`wp-content/themes/` באתר וורדפרס עם התוסף **ACF Pro**.
שדות ה־ACF נשמרים כ־Local JSON תחת `inc/acf-json/` — אין צורך להגדיר ידנית בממשק.
